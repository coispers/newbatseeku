# BatSeekU

BatSeekU is a mobile app prototype built with Expo and Expo Router. It focuses on trusted campus services, tutoring, and errands.

## Getting Started

1. Install dependencies:

```
npm install
```

2. Start the app:

```
npm run start
```

## Scripts

- `npm run android`
- `npm run ios`
- `npm run web`

## Notes

- Uses Expo Router with file-based routes in `app/`.
- Styling is done with React Native StyleSheet.
- Mock data lives in `constants/mock-data.ts`.
