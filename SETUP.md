# BatSeekU Setup

## Prerequisites

- Node.js LTS (check with `node -v`)
- npm (bundled with Node.js)

## Install

```sh
npm install
```

## Run (Web)

```sh
npm run web
```

## Run (Android)

```sh
npm run android
```

## Run (iOS)

```sh
npm run ios
```

## Notes

- If the web dev server asks to switch ports, choose the suggested port.
- If you run into dependency issues, delete `node_modules` and run `npm install` again.
